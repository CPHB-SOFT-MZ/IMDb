# IMDb
Python project for analysing IMDb data

## Prerequisites
- ```bokeh```
- ```numpy```
- ```pandas```

## How to run
- Make sure you have the prerequisites installed
- Clone this project
- In your shell, go to the cloned project
- Type ```jupyter notebook IMDb.ipynb```
